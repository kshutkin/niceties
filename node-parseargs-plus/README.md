# @niceties/node-parseargs-plus

Enhanced wrapper around Node.js built-in [`util.parseArgs`](https://nodejs.org/api/util.html#utilparseargsconfig).

## Installation

```
npm install @niceties/node-parseargs-plus
```

## Usage

```js
import { parseArgs } from "@niceties/node-parseargs-plus";
```
